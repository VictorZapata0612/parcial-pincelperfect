# 🎨 Parcial Pincel Perfect

Este proyecto corresponde al **parcial práctico** de la asignatura **Desarrollo de Aplicaciones Web I** en la **Institución Universitaria Antonio José Camacho**.  
Consiste en la **maquetación de una página web responsive**, inspirada en el sitio oficial de **Disney**, utilizando **HTML5 y CSS3**.

---

## 📌 Descripción del proyecto

El sitio web presenta una landing page moderna que incluye:

- Encabezado con navegación principal
- Banner principal (Hero section)
- Secciones informativas y promocionales
- Diseño responsive para escritorio, tablet y móvil
- Uso de Flexbox y CSS Grid
- Implementación correcta del modelo de caja

El proyecto **no utiliza JavaScript**, ya que el objetivo principal es evaluar la estructura HTML, los estilos CSS y el diseño responsive.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** – Estructura semántica
- **CSS3**
  - Flexbox
  - CSS Grid
  - Media Queries
  - Box Model
- **Diseño Responsive**
- Metodología de clases tipo **BEM**

## 📱 Responsive Design

El sitio se adapta correctamente a distintos tamaños de pantalla mediante **media queries**, cubriendo:

- 🖥️ Escritorio
- 📱 Dispositivos móviles
- 📲 Tablets

---

## 🚀 Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio
2. Verificar que todos los archivos y la carpeta `images` estén presentes
3. Abrir el archivo **`index.html`** en cualquier navegador web

No se requieren dependencias ni instalación adicional.
